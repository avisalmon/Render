# babook.co.il — Scope Definition (Quick Reference)

> **Current strategy note (May 2026):** This quick reference is historical market research. The active Chapter 2 plan is corporate-training authority first: lean `/corporate/` page, WhatsApp/contact conversion, newsletter capture, and one flagship course. Token resale, skill marketplace, advisor marketplace, MCP marketplace, and broad community infrastructure are future options, not the current MVP.

Your 5 scopes, with research findings:

## 1. Training (3 Levels)

### User Level (Beginners)
- **Goal:** Onboard non-technical people to AI basics
- **Example courses:** "ChatGPT 101", "Prompt engineering", "Vibe coding"
- **Effort:** Easy (2-4 hour video + lab)
- **Market:** Large (millions globally, 10K+ in Israel)
- **Competitor:** DeepLearning.AI, Coursera
- **babook edge:** Integrated with token + community

### Practical Level (Practitioners)
- **Goal:** Build production AI tools
- **Example courses:** "GitHub Copilot mastery", "Build AI agents", "LangChain apps", "MCP server dev"
- **Effort:** Medium (6-12 weeks each, hands-on labs)
- **Market:** Medium (100Ks globally, 2-5K in Israel)
- **Competitor:** Kaggle, GitHub Copilot, Replit
- **babook edge:** Community + skill marketplace + token resale

### Deep Level (Researchers)
- **Goal:** Train/fine-tune models from scratch
- **Example courses:** "Transformers from scratch", "Fine-tuning LoRA", "RLHF", "Distributed training"
- **Effort:** Hard (12+ weeks, needs GPU compute)
- **Market:** Small (100s globally, 100-200 in Israel)
- **Competitor:** DeepLearning.AI (advanced), Hugging Face (research)
- **babook edge:** Community mentorship + compute provisioning

## 2. Community

### Forums
- **What:** Q&A + discussion board (like Stack Overflow or Reddit)
- **Effort:** Low (2-3 weeks infrastructure)
- **Retention:** High (keeps users engaged daily)
- **Monetization:** None directly (retention play)

### Skill Marketplace
- **What:** Users create & sell 2-20 hour mini-courses
- **Effort:** Medium (1-2 weeks platform + moderation)
- **Revenue:** 30% commission on creator sales
- **Market:** Growing (creators ≈ 50-500, students ≈ 1-10K)
- **babook edge:** Combines with forums (creators get audience)

### Newsletters
- **What:** Weekly AI digest + monthly deep-dive
- **Effort:** Low (30 min/week writing)
- **Revenue:** Sponsorships (₪5-10K/month at scale)
- **Engagement:** 2-5% open rate (low), but builds brand

### Meetups (Virtual + Physical)
- **What:** Weekly coding sessions, monthly AMA, quarterly conference
- **Virtual:** No logistics, global reach
- **Physical:** Tel Aviv, Jerusalem, Haifa (logistics, but community-building)
- **Revenue:** Ticket sales (₪200-2,000) + sponsorships
- **Effort:** Medium (organizer needed)

### Solutions Hub
- **What:** Curated "how-to" answers for common problems
- **Effort:** Low (50-100 posts by community)
- **Revenue:** None (retention play)
- **Engagement:** High (SEO value, solves pain points)

## 3. Token Provider (Sits on GitHub Copilot)

### GitHub Copilot Resale
- **What:** Buy bulk from GitHub, resell to users at margin (₪149-199/mo)
- **Effort:** Low (1-2 weeks negotiation + integration)
- **Revenue:** ₪30-50/seat/month per 100 users
- **Scale:** Copilot market growing 5-10% MoM

### OpenAI API Credits Store
- **What:** Pre-purchase OpenAI credits at 5-10% discount
- **Effort:** Low (1 week API integration)
- **Revenue:** 5-10% margin on ₪100-1000 bundles
- **Volume:** Depends on practitioner demand

### MCP Server Marketplace
- **What:** Developers publish custom "tools" (integrations), users subscribe
- **Effort:** High (2-3 months infrastructure)
- **Revenue:** 30% commission per MCP subscription
- **Market:** Emerging (few servers today, 100s by 2027)

---

## 4. Service Store (Advisory, Lecturers Hub, Hiring)

### Advisor Marketplace
- **What:** 1-on-1 paid consultations (code review, career advice, project brainstorming)
- **Advisors:** Vetted professionals (GitHub stars, publications)
- **Pricing:** ₪300-1000/hour
- **babook commission:** 20-30%
- **Effort:** High (vetting, reputation system, booking system)
- **Timeline:** Needs 5-10 advisors to launch

### Lecturers Hub
- **What:** Corporate training service (companies hire babook lecturers for on-site/online training)
- **Revenue:** 15-25% commission per deal
- **Effort:** Medium (lead capture + matching + contracts)
- **Timeline:** Slow (B2B sales cycles = 3-6 months)

### Hiring Board
- **What:** Job listings from Israeli AI companies
- **Monetization:** Companies pay ₪2-5K to post
- **Revenue:** ₪10-50K/month at scale (100-200 jobs/month)
- **Effort:** Low (job listing CMS + moderation)

## 5. Emerging Features (Not in Original 5, But Worth Considering)

- **Sponsorships:** Companies sponsor courses for brand + recruitment
